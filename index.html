☰ CSV Format Instructions
  • Save CSV files as "CSV (Comma delimited) (*.csv)" in Microsoft Excel.
  • File must contain a column for emails (e.g., "To", "CC" (Optional), "BCC" (Optional)).
  • Multiple emails can be separated by semicolon (";").
  • First row can optionally contain "Subject" and "Body" (column headers) to embed the template (do not use "\n" for newlines in CSV, use ALT+ENTER).
  • All other columns act as variables (e.g., "Role" becomes "<<Role>>").
